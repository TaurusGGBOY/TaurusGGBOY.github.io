博客的搭建

hexo初始化

安装git

安装nodejs

新建文件夹blog

在blog下运行命令npm -install -g hexo-cli

运行hexo init myblog

cd myblog

npm install

hexo g

hexo server

进入http://localhost:4000就可以看到你的网站了
