## [Java Web 实战乱序1]这里有一个Spring后台项目但是我怎么能够运行呢？

今天，在这里，我们，经过机缘巧合获得了一个Spring后台项目，那么我们到底怎么能够点到那个运行按钮呢？

其实并不简单哈哈哈。

ccc

### 1 导入项目到Idea

看图说话

![](http://q5jmnw63e.bkt.clouddn.com/20200302095416.png)

### 2 安装maven

- 从
  http://maven.apache.org/download.cgi下载apache-maven-3.6.3-bin.zip
- 解压maven到任意文件夹
- https://www.runoob.com/maven/maven-setup.html
  添加
  MAVEN_HOME
  Path
  这两个里的环境变量

### 3 修改settings.xml

- 进入maven 3.6.3根目录的conf文件夹
- 右键记事本/notepad++编辑
- 找到
  <mirrors> .....</mirrors>
- 加入以下代码在 ↑

```xml
<mirror>
    <id>nexus-aliyun</id>
    <mirrorOf>*</mirrorOf>
    <name>Nexus aliyun</name>
    <url>http://maven.aliyun.com/nexus/content/groups/public</url>
</mirror>
```
